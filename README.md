# Course-api

This is a simple course api using nestjs and typeorm.

## Installation

### Docker

```bash
$ brew cask install docker
```

### Package install

```bash
$ pnpm install
```

## Running

### Local development

```bash
$ docker-compose up -d
```

### Domain

- 강사, 수강생, 강의, 수강신청

### Requirements

- 강사는 강의를 생성 및 수정할 수 있다.
- 수강생은 강의를 신청할 수 있다.
- 강의 목록과 상세 정보를 조회할 수 있다.
